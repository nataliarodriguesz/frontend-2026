## 📚 Sobre

Este repositório reúne **todas as atividades, exercícios e trabalhos desenvolvidos em aula** na disciplina de **Frontend – 2026**.

O objetivo é manter uma organização clara do conteúdo produzido ao longo do semestre, documentando a evolução técnica em desenvolvimento Frontend.

---

## 🗂️ Organização das Atividades

As atividades estão organizadas em pastas individuais seguindo um padrão:

```bash
atividade-01/
atividade-02/
atividade-03/
...
```

Cada pasta contém:

* 📄 `README.md` → explicação da atividade
* 🧩 Arquivos do projeto (HTML, CSS, JS etc.)
* ▶️ `index.html` (quando aplicável)
